# 🎮 Task 03 – Advanced Quiz Web Application  
Web Development Internship – SkillCraft Technology  

---

## 📌 Project Overview  

This project was developed as part of Task 03 of the Web Development Internship at SkillCraft Technology.

It is a dynamic quiz application powered by the Open Trivia Database API. The application includes category selection, difficulty selection, timer functionality, progress tracking, and a local leaderboard system.
